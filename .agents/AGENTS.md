# Auto-Commit and Push

- **Always Push Changes**: After making any file changes (e.g., code edits, configuration updates), automatically commit those changes with a descriptive commit message and push them to the remote GitHub repository. Do not wait for the user to prompt you to push.

# Daily Naukri Role Check

- **Daily Naukri Role Check**: At the start of a new chat or conversation in this workspace, proactively use the web search tool to find currently trending or available job titles/roles related to "DevOps Engineer" on Naukri.com. Extract any new relevant roles (e.g., specific variations of Cloud or Platform Engineer) and add them to the `SEARCH_KEYWORDS` list in `config.py`. Follow up by committing and pushing the updates.
